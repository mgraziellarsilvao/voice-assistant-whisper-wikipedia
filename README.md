# 🎤 Voice Assistant with Whisper + Wikipedia

## 📌 Overview
This project is a voice assistant built in Python that converts speech into text using Whisper and retrieves intelligent responses from Wikipedia.

## 🚀 Features
- 🎤 Speech-to-text with Whisper
- 🌍 Intelligent responses via Wikipedia
- 🔊 Text-to-speech (gTTS)
- 🧠 Offline-first architecture

## 🧰 Technologies
- Python
- Whisper (OpenAI)
- Wikipedia API
- gTTS
